RTX Video Converter Portable
============================

This folder is the no-install portable build of RTX Video Converter.

How to use
1. Keep all files in this folder together.
2. Double-click rtx-video-converter.exe to start the app.
3. The app will automatically launch the bundled vsr_backend.exe sidecar.

Notes
- This package still depends on Microsoft Edge WebView2 being available on Windows.
- RTX Video features require a supported NVIDIA GPU and the matching runtime DLLs included beside the app.
- If Windows SmartScreen prompts on first launch, choose the appropriate allow/run option for your environment.

Files you should not remove
- rtx-video-converter.exe
- vsr_backend.exe
- All .dll files in this folder
